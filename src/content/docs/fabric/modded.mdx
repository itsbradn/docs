# Get started on braden's modded server

MAKE SURE TO HAVE JAVA INSTALLED!

[https://www.java.com/en/download/](https://www.java.com/en/download/)

## Windows

Only for windows apps.

### Install Fabric

Download fabric from [here](https://maven.fabricmc.net/net/fabricmc/fabric-installer/0.11.2/fabric-installer-0.11.2.exe) and run the exe.

You should see the following

![Installer](/img/fabric_installed_first.png)

Change Minecraft Version to 1.19.4

![Installer](/img/fabric_installed_second.png)

Click install and wait for it to install.

![Installer](/img/fabric_installed_third.png)

### Download mods

Download the zip file [here](https://drive.proton.me/urls/N9AFF6AF48#tMvr75a6HqBN).

### Install mod pack

Open your file explorer to This PC

![Install](/img/install_modpack_first.png)

Enter `%appdata%/.minecraft/mods` into the directory section of your file explorer

![Install](/img/install_modpack_second.png)

Move the downloaded zip file into this folder and extract it

![Install](/img/install_modpack_third.png)

Verify extract location only is `/mods` not `/mods/mods` then click extract.

![Install](/img/install_modpack_fourth.png)

You can now exit your file explorer and open minecraft.

After opening minecraft change version to fabric 1.19.4 and start the game

You can now ask braden for the IP and connect to the server.
